Example Page

This page demonstrates loading the navigation menu from the microservice endpoint.

How it works:

  1. Navigation menu is served from /nav endpoint
  2. JavaScript fetches and injects the HTML component
  3. The component includes mobile-responsive hamburger menu
  4. All pages can load the same navigation microservice

API Endpoints:

  • → GET /nav - HTML component
  • → GET /api/navigation - JSON data
